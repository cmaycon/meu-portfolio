Portfólio online desenvolvido como currículo interativo para apresentar minhas experiências profissionais, formação acadêmica, habilidades técnicas e certificações.

🌐 **Acesse online:** [cmaycon.github.io/meu-portfolio](https://cmaycon.github.io/meu-portfolio/)

---

## 🧠 Sobre o projeto

Este portfólio foi criado com o objetivo de reunir em um só lugar todas as informações relevantes da minha carreira como desenvolvedor. A proposta é ter um currículo vivo, acessível de qualquer lugar e que reflita minha identidade profissional.

A ideia surgiu da necessidade de um espaço pessoal onde recrutadores e parceiros pudessem conhecer minha trajetória de forma rápida, visual e interativa.

---

## ✨ Funcionalidades

- 🧭 Navegação suave entre seções
- ⌨️ Efeito de digitação no título (typewriter)
- 💼 Timeline de experiências profissionais
- 🎓 Formação acadêmica com status de curso atual
- 📜 Certificações relevantes
- 🧰 Skills divididas por categorias
- 📱 Layout totalmente responsivo (mobile, tablet, desktop)
- 🔗 Links diretos para redes sociais e contato
- 📄 Download do currículo em PDF
- 🎨 Design minimalista com toques de código (estilo dev)

---

## 🚀 Tecnologias utilizadas

| Tecnologia | Uso |
|------------|-----|
| **HTML5** | Estrutura semântica do conteúdo |
| **CSS3** | Estilização, animações e responsividade |
| **JavaScript** | Interatividade, menu mobile e efeito typewriter |
| **Google Fonts** | Tipografia Inter e Fira Code |
| **Font Awesome** | Ícones das seções e redes sociais |
| **GitHub Pages** | Hospedagem gratuita do site |

Nenhum framework ou biblioteca externa — apenas HTML, CSS e JS puros.
